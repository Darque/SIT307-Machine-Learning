Weekly wrap up

Congratulations, you have completed Week 1!

This week was about understanding machine learning very broadly. We have looked at its practical uses, different sub types and how they are evaluated. You also got the opportunity to play with Python code. You will use Python to code most of the algorithms during this unit so it’s important to put each piece of code into Jupyter Notebook and run the code.

Next week, we will continue exploring Python programming by using different modules and packages. We will also start exploring certain fundamental concepts in linear algebra such as vectors, matrices and operations on them. Getting these concepts clear in your mind is important as they are fundamental to machine learning algorithms.

We look forward to exploring these ideas with you next week.
