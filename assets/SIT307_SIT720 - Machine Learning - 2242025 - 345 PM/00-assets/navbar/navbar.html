Page Edit-Mode
Do you wish to load the navBar in page EDIT MODE - optional (true/false):

false

Hint: This field determines whether the navbar is loaded in EDIT MODE in a page in which it is embedded (not this page!)
NOTE: If you opt to show the navBar in page EDIT MODE, there will be a minimal lag in loading the page in edit mode in the order of 0.01-0.3 seconds. If you opt not to show it, you will see a place-holder (with the same dimensions) instead of the navbar
Default value: If this field is empty, improperly formatted or has a bad value, its default value is false.
Available values: true or false
Variable Reference: userLoadNavEditMode


NavBar Wrapping
Allow wrapping - optional (true/false):

true

Hint: When set to true, allows wrapping of the navBar once the window size reaches a small enough value
Default value: If this field is empty, improperly formatted or has a bad value, its default value is true.
Available values: true or false
Variable Reference: userAllowWrapping


Included Resources
The types of topics to include in the navbar - optional (one of 'any', 'file' or 'html'):

html

Hint:This field sets the types of resources to include in the navbar, and can be one of 'any', 'file' or 'html'. Note that setting it to anything other than 'html' means that the user will lose the navBar in the other activity types once they click on that activity type within a html page with the navBar included.
html: Shows only html pages listed within your module as navBar elements
file: Shows only html pages AND files (e.g. pdf files) listed within your module as navBar elements
any: Shows only html pages, files (e.g. pdf files) and other activity types such as discussions, submission folders etc listed within your module as navBar elements
Default value: If this field is empty, improperly formatted or has a bad value, its default value is 'html'.
Available values: any, file or html
Variable Reference: userIncludedResources

Hint:This field sets whether or not you want to include the 'Back' and 'Next' buttons at either end of the navbar.
Default value: If this field is empty, improperly formatted or has a bad value, its default value is true.
Available values: true or false
Variable Reference: userIncludeBackNextBtns
